まず本題入る前にConsole画面の出し方....

やっべぇ！Console画面の開き方忘れた！という方に....

ここでゆっくり解説していきましょ！

コンソールとは各ブラウザに実装されており、動作確認、デバック等に活用する時に使われております。

ここではブラウザGoogleChromeを例に使っていきたいと思います。

ではまずGoogleChromeを開いてみましょう。

開いたら適当はWebページを開いて、そこで CTRL + SHIFT + I (もしくはF12キーのみ)を押してみてください。

すると。。。

右側にこんな感じの画面が出てきましたね？

これは開発者ツールと言い、いわゆるページのソース確認が出来ます。

ここで赤い枠にあるConsoleとありますが、これがコンソールになります！

Consoleボタンを押すとコンソール画面に行きます。

ではでは本題のコンソールオブジェクトへ。。。

よく使われるConsoleオブジェクト

Consoleは極めてますが、log以外あるの？と思っていますね？

実はlog以外にも沢山あるようです。

沢山種類がありますが、今回は簡単に使う(よく使うと言われている)オブジェクトを4つ紹介します。

警告アイコン

    <script>

    console.warn("文章");

    </script>

これは、Console画面に警告のアイコンが付きます。

エラーアイコン

    <script>

    console.error("文章");

    </script>

これは、Console画面にエラーのアイコンが付きます。

複数のオブジェクト表示

    <script>

    console.info("文章1" , "文章2");

    </script>

Console.logは単一のオブジェクトを表示しますが、infoは複数含めて表示する事が出来ます。

コンソールでカウント

    <script>

    var kansu = new Array("まぐろ","さーもん");
 
    //ここで１次関数作成

    for(i = 1 ; i < kansu.length; i++){   //ループfor分

      console.count("文字名",kansu);

    } 

    </script>

これは、一次配列にされている関数の個数を数えます。

※一次配列に格納されている文字を表示しません。個数のみです。

その他のConsoleオブジェクト集

メインによく使うlog,warn,error,info,countを紹介しましたが、その他のオブジェクトを紹介します。

Console.clear()

コンソールをクリアします。

Console.table()

表形式のデータを、テーブルで表示します。

Console.debug()

デバックを表示しますが、Chromeでは非推奨と言われています。

Console.dir()

オブジェクトのメソッドを階層表示で確認できます。
