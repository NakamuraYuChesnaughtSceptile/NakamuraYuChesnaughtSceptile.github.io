・使い方 + Github-pagesを作ろう！

まずはリポジトリを作成してみましょう！

Githubにアカウント登録後、ログインします。

次に下の画像通りにある、New Repository(赤い枠)を押します。↓



クリックすると、作成の画面が表示されます。色々ありますが１つずつ説明していきましょう。



まず最初はRepository nameとありますがここは [ 自分のユーザー名 ] + github.io を入力してください！

名前を入力したら次はDescriptionですが意味は「説明文」で適当に入力してくたさい。

次はPublic(公開)かPrivate(非公開)とありますが、ここは無料会員の方はPublicしか選択できません。

Privateにしたい方は有料会員になってね(* ͡° ͜ʖ ͡°*)

最後にリポジトリの中にREADMEファイルを作成したい方は(Initaialize this Repository with a README)にチェックを入れてください。

後はCreate repositoryを押せば、作成完了です！

・ファイルを作成！編集！

ここではVSC(Visual Studio Code)ツールを使っての説明をしていきます。

はじめに適当にhtmlファイルを作成します。(中身bodyは)hello world等書いておけばいいでしょう。

そしたらターミナルコードを開きコードを入力します。今回は[index]という名前で作成します。

mkdir index

mkdir とはディレクトリを新しく作るコマンドです。これで作成が出来ます。

cd index

cd とは次へ開くフォルダを示し、indexというフォルダを開きます。

git init

git init でindexという名のフォルダにGitリポジトリが作成されます。

これでこのフォルダはgithubと連携する事が出来ました！

・作った奴をフォルダに追加＆アップロード！


おぉ！作れた！というなら、じゃぁこれをフォルダに追加しましょ！

git add index.html

はい、これでもう追加おｋなんです！簡単でしょ？

ささ、次はレポジトリにコミットさせてみましょう！

git commit -m "add new file"

これで、リポジトリでのファイルの追加が出来ました！

じゃあ確認してみましょ！

git status

statusで今何のブランチ入っているかを確認出来ますん。

ブランチ名が master になっている事を確認したら最後にアップロードします。

git push

これでアップロードは完了です！。サイトにいって確認してみましょう。



ありました！ちゃんとアップロードされています！

最後にブラウザを開き、URLに ユーザー名 + .github.io と入力し、index.html内に入力したhello worldが表示されたら成功です。

更新の仕方は、そのまま同じファイルをアップロードすれば自動上書きが出来ます。

・今回使ったコマンド一覧まとめ + α

mkdir 名前 新しくディレクトリ作成
cd 名前 名前のファイルを開く
git add 名前.html 名前.htmlを作成
git init Gitリポジトリ作成(隠しフォルダになります。)
git add 作成したデータを追加
git commit -m "add new file" リポジトリでのファイルの追加
git status ブランチ確認
git push アップロード
