window.open() とは

直訳で言いますと、窓オォ～プン♪

いわゆる、別ウィンドウが開きます。

今回はクリックすると別ウィンドウが開くという設定を作ります。

完成はこんな感じです。

ではではコードを入力していきましょぉ～　＋解説もねっ

HTML

    

    <body>

    <form>

    <input type="button" value="てすと" onclick="subwindow1()">

    </form>

    </body>

javascript

      

    <script>

      function subwindow1(){

        window.open( 'URL', 'ウィンドウ名' ,'オプション名' );

      }

    </script>

HTMLから解説していきます。

まずformでinputボタンを作成します。valueはボタンの名前です

onclickはクリックした時のイベント名を動かします。今回はsubwindow1という名前で設定します。

続いてめっちゃ記述してるjavascriptを見てみましょう

まずその時のイベント名 function 名前{}で定義します。

{}内もゆっくりみてみましょう。

　　window.open( 'URL', 'ウィンドウ名' ,'オプション名' );

URLはページ先のリンク。ウィンドウ名はそのウィンドウの名前です。

オプションは細かく設定できます。width,heightなど

※menubar,toolbar,location,status,resizable,scrollbars

これらは最近のブラウザでは設定しても無視されるようになりました。

記述の仕方はHTMLの記述と同じです。(例：<img src="" width="200px",height="200px">)

window.close() とは

そのままの意味で、ウィンドウを閉じます。

下のボタンでやってみてください。

なりましたね？それでは確認してみましょう。

HTML(サブウィンドウの方)

      

    <body>

    <form>

    <input type="button" value="close" onclick="subwindow2()">

    </form>

    </body>

javascript(サブウィンドウの方)

        

    <script>

      function subwindow2(){

        window.close();

      }

    </script>

HTMLはwindow.openの時とほぼ同じです。

javascriptはwindow.close() たったのこれだけでおｋです。

window.focus() とは

フォーカスとは指定のウィンドウに手前に表示させる事をいいます。

実際にサブウィンドウを開いて、フォーカスボタンを押してみましょう！

どうです？フォーカスボタンを押しましたが何も起こりません。

何故なら、もう手前に表示されているため何も起こりません。

と、とりま、コード確認してみましょ。

javascript(サブウィンドウの方)


    <sctipt>

    function subwindow3(){

      window.focus();

    }

    </script>

これでおｋです。
