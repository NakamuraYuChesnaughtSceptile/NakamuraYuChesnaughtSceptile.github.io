Githubにアップロードの仕方

まずGitbubサイトでrepositoryを作成するが、作成する名前は自分の　ユーザー名　+　.github.ioで作成をする。
作成したら、VSCodeを起動、ターミナルコードに git clone URL名を入力し、フォルダを作成する。
そのフォルダの中にhtmlを作成、作成したら適当に入力する。 作成したらgit statusと入力し、中身がorigin masterの中に入ってる事を確認する。
違うブランチに入っていたら、[git checkout master]とコマンドに入力する。　そして[git status]と再入力し、中身を確認
中身を確認したらアップロードをする、アップロードの仕方は[git push -u origin master]と入力しアップロードをする
アップロードしたら、ブラウザを開き、URLに ユーザー名 + .github.io と入力し、表示されたら成功。
